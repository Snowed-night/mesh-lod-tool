# 课程大作业提交说明

题目：误差驱动的三维网格 LOD 简化与可视化系统

## 1. 提交材料总览

本项目按照课程要求拆分为“代码 + 文档 + PPT”三类材料。其中 PPT 后续单独制作，当前已准备 PPT 内容提纲。

正式提交时优先使用 `report/word/` 目录下的 Word 文档。Word 文档已统一设置为：中文宋体，英文 Times New Roman，正文小四。

代码（含注释）：

```text
https://github.com/Snowed-night/mesh-lod-tool
```

代码仓库包含：

- `src/`：C++ 网格读写、QEM 基线简化、meshoptimizer 后端、OpenGL 查看器。
- `web/`：模型上传、3MF 转 OBJ、LOD 生成、Web 可视化、光照控制、误差热力图。
- `third_party/meshoptimizer/`：工程化网格简化依赖。
- `README.md`：项目介绍、构建命令、运行方式和复现说明。
- `report/`：课程报告、素材清单和 PPT 提纲。
- `result/report_assets/composite/`：最终报告使用的合成效果图。

## 2. 三份平时作业对应关系

```text
report/02_technical_report.md
```

内容重点：

- 项目目标、输入输出和系统结构。

- 真实模型实验统计和结果图。

平时作业 3：未来工作展望、个人感想与思政报告

```text

```

内容重点：
