## 图 1 到图 3：三模型 LOD 对比大图

目标：生成 3 张论文式网格图，每张图对应一个模型。Indoor Plant 和 Girl 每张图包含 20 张子图，Horse 因未截图 PBR 模式，最终为 15 张子图。

布局规则：

```text
Indoor Plant / Girl = 4 行 x 5 列
Horse = 3 行 x 5 列
行：线框、无材质、真实材质、热力图；Horse 不包含真实材质行
列：LOD 100、LOD 075、LOD 050、LOD 025、LOD 004
```

建议统一使用自然光，视角、缩放、模型位置尽量保持一致。这里按系统实际生成的 `LOD 004` 作为最低展示档。

## 图 5：Girl 自适应 LOD

目标：展示不同屏幕误差阈值下，从近到远时自适应 LOD 的切换效果。最终合成一张 4 x 3 图。

布局规则：

```text
行：近景、中近景、中远景、远景
列：低阈值、中阈值、高阈值
```

截图保存到：

```text
result/report_assets/raw/adaptive/girl/threshold_low/
result/report_assets/raw/adaptive/girl/threshold_mid/
result/report_assets/raw/adaptive/girl/threshold_high/
```

每个阈值文件夹内保存：

```text
view_01_near.png
view_02_mid_near.png
view_03_mid_far.png
view_04_far.png
```

最终合成图输出为：

```text
result/report_assets/composite/fig05_girl_adaptive_lod.png
```

建议阈值：

```text
threshold_low：1.0 px
threshold_mid：3.0 px
threshold_high：6.0 px
```

如果实际效果不明显，可以改成 `0.5 px / 2.0 px / 8.0 px`。关键是三列的当前 LOD 和屏幕误差要能看出差异。

## 视频 1：最终演示视频

只录一个综合视频，保存到：

```text
素材/截图或者视频/final_demo/video01_final_demo.mp4
```

建议结构：

```text
1. 同一模型，自然光，展示手动 LOD 的无材质和线框。
2. 录到右侧误差曲线和 LOD 参数变化。
3. 调整屏幕误差阈值，展示自适应 LOD。
4. 调整热力图强度，展示真实材质和热力图。
5. 展示不同角度、颜色的自然光。
6. 展示不同位置、颜色的点光源。
```

建议时长：60 到 90 秒。太短讲不清系统能力，太长答辩时不容易播放。

## 截图统一要求

1. 同一组对比图必须保持相同窗口大小、相同自然光、相同相机视角。
2. 图 1 到图 3 尽量不要把左侧上传区截进去，只截模型视窗和必要的右侧参数即可。
3. 热力图不要整片红黄，调到能看到蓝、黄、红过渡。
4. 截图中不要出现“等待上传”、“处理失败”等无效状态。
5. 文件名必须完全按本清单保存，后续才能自动合成。
6. 优先用 PNG，不要用 JPG，避免线框和文字被压缩糊掉。
